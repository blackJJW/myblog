- **기본 자료형**(**primitive types** 또는 **primitives**) : 자바스크립트에서 **실체가 있는 것(undefined와 null 등이 아닌 것) 중에 객체가 아닌 것**
    - ex) **숫자**, **문자열**, **불**
- 이러한 자료형은 **객체가 아니므로 속성을 가질 수 없음**
