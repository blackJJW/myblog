- 배열 내부에 들어 있는 **요소의 개수를 확인**할 때는 배열의 **length 속성**을 사용
- 배열의 인덱스는 **0부터 시작**
    
    ```jsx
    배열.length
    ```
    
    ![Untitled](/images/lang_javascript/study/JavaScript_배열_요소_개수_확인/Untitled.png)
    
    ```jsx
    배열[배열.length - 1]
    // 배열의 마지막 요소를 선택할 때 자주 사용
    ```
