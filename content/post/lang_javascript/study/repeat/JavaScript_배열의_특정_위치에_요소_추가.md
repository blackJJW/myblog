- **배열의 특정 위치(인덱스)**에 요소를 추가할 때는 **splice() 메소드를 사용**
- splice() 메소드의
    - 2번째 매개변수 : 0을 입력하면 아무것도 제거하지 않음
    - 3번째 매개변수 : 추가하고 싶은 요소를 입력
    
    ```jsx
    배열.splice(인덱스, 0, 요소)
    ```
